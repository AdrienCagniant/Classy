# Pasteboard

```4d
pasteboard:=classy.Pasteboard.new()
```

## Text

```4d
$text:=pasteboard.getText()
```

```4d
pasteboard.setText($text)
```

## File

```4d
$file:=pasteboard.getFile() // File instance
```

```4d
pasteboard.setFile($file)
```

## Picture

```4d
$picture:=pasteboard.getPicture()
```

```4d
pasteboard.setPicture($picture)
```
